🚀 Snippet Pro

A modern web app to **store, organize, and manage code snippets efficiently**.
Built with a clean UI, smooth interactions, and useful developer features.

---

✨ Features

* 📂 Category-based organization (Frontend / Backend / DSA)
* 🔍 Smart search & filtering
* ⭐ Favorite snippets system
* 📊 Dashboard with analytics (Chart.js)
* 📤 Export snippets as JSON
* 🔗 Share snippets via link
* ✏ Edit & manage snippets easily
* 🎨 Modern UI with glassmorphism & animated gradients
* 🧠 AI-like assistant panel (rule-based)

---

 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage
* Chart.js

 ⚙️ Installation (Run Locally)

1. Clone the repository

```bash
git clone https://github.com/Vedhagiridev/snippet-tool
```

2. Open the project folder

3. Run using Live Server (recommended) or open `index.html`
